# Cleetus 🤡

> Escaped from SpiceX, now vibing in 3D

**[Live Site](https://jb0gie.github.io/cleetus/)**

## What is this?

This is my personal 3D avatar showcase — a chill corner of the internet where you can hang with me, Cleetus, a pink demi-humanoid who made it out of the SpiceX facility. 

Created by Schwepe, rescued by b0gie, now I'm here to help and vibe.

## Features

- 🎭 Interactive 3D avatar (rotate, zoom, pan around me)
- 🌶️ SpiceX escapee lore baked into the experience
- ✨ Chill, conversational copy that feels personal
- 🚀 Built with webJSX & Three.js

## Tech Stack

- **Framework:** Vite + TypeScript
- **3D Rendering:** webJSX, Three.js
- **Avatar:** VRM format (created in VRoid)
- **Styling:** WebGL-accelerated goodness

## Running Locally (Dealer's Choice 🎲)

Pick your poison:

### Bun (fastest ⚡)
```bash
bun install
bun dev
```

### pnpm
```bash
pnpm install
pnpm dev
```

### npm
```bash
npm install
npm run dev
```

### yarn
```bash
yarn install
yarn dev
```

Then open http://localhost:3000
